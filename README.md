# PlantUML → Draw.io Converter

A lightweight JavaScript library to convert PlantUML diagrams into draw.io (mxGraph XML).

## Features
- Actor, Database, Queue support
- Arrow types: -> and -->
- DAG-based layout engine (left-to-right)
- Lightweight and extensible

## Usage

```js
import { convertPlantUMLToDrawIO } from "./src/index.js";
